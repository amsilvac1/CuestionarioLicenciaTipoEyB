# Cuestionario de Licencia de Conducir EC

Aplicación web interactiva para la práctica de preguntas del examen teórico de la licencia de conducir tipo B y E en Ecuador.

## Descripción

Este proyecto permite a los usuarios simular un examen de licencia de conducir, respondiendo preguntas reales y visualizando su puntaje final. Incluye imágenes ilustrativas, retroalimentación inmediata y un diseño moderno y responsivo.

## Características

- Simulación de cuestionario para licencias tipo B (particular) y E (profesional)
- Preguntas con imágenes y opciones múltiples
- Resultados automáticos y visualización de puntaje
- Interfaz intuitiva y adaptable a dispositivos móviles
- Desarrollado con Vue 3 y Vite para alto rendimiento

## Instalación y uso

1. Clona este repositorio:
   ```bash
   git clone <url-del-repositorio>
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Accede a la aplicación en [http://localhost:5173](http://localhost:5173)

## Estructura del proyecto

- `src/components/` — Componentes reutilizables (tarjeta de pregunta, resultados, etc.)
- `src/data/` — Preguntas y datos del cuestionario
- `src/views/` — Vistas principales de la aplicación
- `public/imagenes/` — Imágenes asociadas a las preguntas

## Tecnologías utilizadas

- [Vue 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
